停停手机（StopMobile）隐私政策

最后更新日期：2026年2月10日

生效日期：2026年2月10日

感谢您使用停停手机（以下简称“本应用”或“我们”）。我们非常重视您的隐私权，尤其是涉及儿童使用的场景。本隐私政策旨在向您清晰、透明地说明本应用如何收集、使用、存储和保护信息。

本应用是一款纯本地运行的家长控制工具，专为帮助家长管理儿童设备使用时间而设计。我们致力于最大限度地保护用户（包括儿童）的隐私，所有功能均基于 Apple 官方提供的 Family Controls、Device Activity 和 Screen Time API 实现。

1. 我们收集的信息

本应用不收集任何个人信息。

具体说明：

  • 本应用不要求用户注册账号，无需提供姓名、邮箱、电话、地址等任何个人身份信息。
  • 本应用不联网，不向任何服务器上传数据。
  • 所有数据（包括设备使用时长、应用使用记录、白名单设置、限制时段、订阅状态等）仅存储在您的本地设备上，仅供家长在设备本地查看使用报告。
  • 我们不使用任何第三方分析工具、广告服务或跟踪技术。
  • 订阅功能完全通过 Apple 的 StoreKit 框架处理，支付信息由 Apple 直接管理，我们无法访问您的支付细节。

2. 我们如何使用信息

由于本应用不收集任何个人信息，因此不存在远程使用或处理用户数据的情况。

本地生成的数据仅用于以下目的：

  • 实现家长控制功能（如应用白名单、时段限制、儿童模式倒计时、延长请求等）。
  • 生成本地使用报告，帮助家长了解儿童设备使用情况（报告仅在设备上显示）。
  • 管理订阅状态（试用期、订阅有效期），以控制功能可用性。

所有数据处理均在您的设备本地完成，不会离开设备。

3. 信息共享与披露

我们绝不与任何第三方共享您的信息。

因为本应用不收集任何可识别个人身份的信息，也不存在上传或传输数据的行为，因此无需与第三方共享数据。

唯一例外情况：

  • 订阅购买通过 Apple App Store 完成，相关交易信息由 Apple 处理并受 Apple 隐私政策约束。
  • 若依法要求（如法院命令），我们将配合提供信息，但由于我们不持有任何用户数据，此类情况实际不会涉及用户隐私。

4. 数据存储与安全

所有数据仅存储在您的 iOS 设备本地，使用 Apple 提供的安全存储机制（如 Keychain 用于密码、UserDefaults 和本地数据库用于设置）。

我们采用了以下措施保护本地数据：

  • 核心设置（如密码、白名单）受 6 位数字密码保护。
  • 利用 Apple Family Controls 和 Screen Time API 的系统级安全机制，确保限制无法被轻易绕过。
  • 应用遵循 Apple 最高安全标准开发，不包含任何恶意代码。

尽管我们已采取合理措施保护本地数据，但请注意，任何设备均存在物理访问风险，建议家长妥善保管设备。

5. 儿童隐私

本应用专为家长管理儿童设备使用而设计，目标用户群体包括儿童（由家长监督使用）。

我们严格遵守《儿童在线隐私保护法》（COPPA）及其他相关法律法规：

  • 本应用不直接向儿童收集任何信息。
  • 所有功能操作均由家长控制（需密码验证）。
  • 设备使用数据仅在本地存储，仅供家长查看，帮助培养儿童健康使用习惯。
  • 我们不以任何方式针对儿童进行广告或营销。

6. 第三方服务

本应用不集成任何第三方 SDK、服务或框架（除 Apple 官方 API 外）。

订阅功能完全依赖 Apple App Store 的官方支付系统，不涉及第三方支付或分析服务。

7. 您的权利

由于本应用不收集个人信息，您无需行使删除、导出或更正等权利。

您可以随时：

  • 在应用设置中清除本地数据（重置白名单、时段等设置）。
  • 卸载应用，所有本地数据将被自动删除。
  • 通过 App Store 管理订阅（取消、恢复购买等）。

8. 本隐私政策的变更

我们可能不时更新本隐私政策以反映功能变更或法律要求。更新后将在本页面公布新版本，并更新“最后更新日期”。

建议您定期查看本政策。继续使用本应用即表示您接受更新后的隐私政策。
